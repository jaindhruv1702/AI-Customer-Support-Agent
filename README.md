# AI Customer Support Agent

## Overview

This project automates customer support operations using AI-powered email classification, knowledge retrieval, and response generation.

The workflow receives incoming customer emails, classifies the request type, retrieves relevant information, generates contextual responses using Gemini, and automatically sends replies.

## Problem Statement

Customer support teams spend significant time handling repetitive inquiries. This project reduces manual effort by automating email triaging and response generation.

## Workflow Architecture

1. Incoming Email Trigger
2. Email Classification Agent
3. Knowledge Retrieval
4. AI Response Generation
5. Automated Reply
